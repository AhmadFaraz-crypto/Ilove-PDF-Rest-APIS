## Features

<dl>
  <dt>PDF to JPG</dt>
  <dd>Convert each PDF page into a JPG.</dd>
  <dt>JPG to PDF</dt>
  <dd>Convert image to PDF in seconds.</dd>
  <dt>HTML to PDF</dt>
  <dd>Convert webpages in HTML to PDF. Copy and paste the URL of the page you want and convert it to PDF with a click.</dd>
</dl>
